═══════════════════════════════════════════════════════════════
  LIGHT THEME — Dhruvi Mittal Portfolio
  Complete Color & Styling Reference
═══════════════════════════════════════════════════════════════

This document describes every color, variable, effect, and stylistic
choice used in the light theme of dhruvi-portfolio.

All values are the light-mode defaults. Dark mode overrides these via
the .dark class on <html>.

─── 1. CSS CUSTOM PROPERTIES (defined in @theme inline) ─────

  --color-background:       #fafafa     (Near-white off-white)
  --color-foreground:       #111111     (Near-black for primary text)
  --color-secondary:        #666666     (Muted gray for body/secondary text)
  --color-border:           #e5e7eb     (Light gray for borders)
  --color-accent:           #2563eb     (Blue-600 — primary accent)
  --color-accent-hover:     #1d4ed8     (Blue-700 — accent hover)
  --color-accent-light:     #eff6ff     (Very light blue tint)
  --color-card:             #ffffff     (Pure white card background)
  --color-muted:            #f5f5f5     (Light gray for muted backgrounds)
  --color-success:          #059669     (Emerald-600 for success states)

  --font-sans:              var(--font-geist-sans)
  --font-mono:              var(--font-geist-mono)

  --radius-sm:              0.375rem
  --radius-md:              0.5rem
  --radius-lg:              0.75rem
  --radius-xl:              1rem
  --radius-2xl:             1.25rem

─── 2. AMBIENT BACKGROUND GLOW (body::before) ───────────────

  Two subtle radial gradients overlaid on the body:

    1. Blue glow at top-left:
       radial-gradient(ellipse at 20% 0%, rgba(37,99,235,0.04) 0%, transparent 60%)

    2. Green glow at bottom-right:
       radial-gradient(ellipse at 80% 100%, rgba(5,150,105,0.03) 0%, transparent 60%)

─── 3. INTERACTIVE DOT BACKGROUND ────────────────────────────

  An HTML5 Canvas draws a dot grid at 16px spacing.

  Light mode settings (dark mode interaction is disabled in light):
    - Base dot radius:      0.6px
    - Idle opacity:         0.04  (rgba(0,0,0,0.04))
    - Dot color:            "0,0,0"  (black)
    - Interaction:          DISABLED (dots remain static in light mode)
    - Mouse radius:         120px  (only used in dark mode)
    - Max opacity (hover):  0.5    (only used in dark mode)
    - Max scale (hover):    2.5    (only used in dark mode)
